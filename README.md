# NexLib

This c++ library allows to control Nextion displays.

## Overview

The goal of this library is to provide the most reliable, secure and faster way to complete control Nextion Displays from a high end c++ application.

## Getting Started

### Prerequisites

This library needs include the "serialib.h" to control serial ports.
